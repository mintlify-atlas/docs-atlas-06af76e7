## Reasoning Models

STRIDE GPT leverages reasoning models that "think" before responding:

<Note>
**What are Reasoning Models?**

Reasoning models use extended internal processing (often called "thinking" or "chain-of-thought") before generating their final response. This leads to more thorough threat analysis but takes longer.
</Note>

### Available Reasoning Models

**OpenAI GPT-5 Series**
- `gpt-5.2` - Best for coding and agentic tasks
- `gpt-5.2-pro` - Smarter, more precise responses
- `gpt-5`, `gpt-5-mini`, `gpt-5-nano` - Varying speed/quality tradeoffs

**Anthropic Extended Thinking**
- Any Claude 4.5 model with "Enable Extended Thinking" checkbox
- Allocates 16K tokens for internal reasoning

**Google Thinking Mode**
- Gemini 2.5 and Gemini 3 models automatically use thinking capabilities
- 1024 token thinking budget

**Groq Reasoning**
- `deepseek-r1-distill-llama-70b` - Fast inference with reasoning capabilities

**Mistral Magistral**
- `magistral-medium-2509`, `magistral-small-2509` - Reasoning-focused models

<Warning>
Reasoning models take significantly longer to respond (30-90 seconds) but produce more comprehensive threat analysis. For faster results, use standard models like GPT-5-nano, Claude Haiku, or Gemini Flash.
</Warning>

## Token Limits & Context Windows

Understanding context windows helps when analyzing large applications:

### Large Context (500K+ tokens)
- **Google Gemini 3**: 1M tokens - Best for massive GitHub repos
- **Google Gemini 2.5**: 1M tokens - Extensive codebase analysis
- **OpenAI GPT-4.1**: 1M tokens - Long conversations and large applications
- **OpenAI GPT-5 series**: 400K tokens - Comprehensive threat modeling

### Medium Context (100K-200K tokens)
- **Anthropic Claude 4.5**: 200K tokens - Detailed security analysis
- **Mistral Large**: 128K tokens - Solid for most applications
- **Groq models**: 128K tokens - Fast processing of moderate-sized apps

### Considerations

<Note>
STRIDE GPT automatically configures token limits based on the selected model and provider. You can adjust these in Advanced Settings if needed.
</Note>

## Cost Considerations

### Premium Tier ($$)
- **OpenAI GPT-5 series**: Highest quality, advanced reasoning
- **Anthropic Claude Opus 4.5**: Most capable Claude model
- **Google Gemini 3 Pro**: Preview pricing, large context

### Standard Tier ($)
- **OpenAI GPT-5-mini/nano**: Fast reasoning at lower cost
- **Anthropic Claude Sonnet/Haiku 4.5**: Balanced performance
- **Mistral models**: Competitive European alternative
- **Google Gemini 2.5 Flash**: Fast and cost-effective
- **Groq**: Fast inference, pay-per-token

### Self-Hosted (Free)
- **Ollama**: Free, requires local compute resources
- **LM Studio**: Free, desktop application

<Warning>
Self-hosted models require significant computational resources (GPU recommended) and setup expertise. Cloud providers offer easier setup and predictable performance.
</Warning>
